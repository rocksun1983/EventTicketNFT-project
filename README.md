EventTicketNFT

A decentralized NFT-based event ticketing platform built on Ethereum that enables event organizers to create events, mint NFT tickets, and allows users to securely purchase, own, and verify tickets on-chain.

🚀 Features
Create and manage events on the blockchain
Mint NFT tickets for events
Purchase event tickets securely
Verify ticket ownership on-chain
Prevent ticket duplication and fraud
Connect with MetaMask wallet
Ethereum smart contract integration
Web3-powered decentralized application (DApp)
🛠️ Technology Stack
Ethers.js
Blockchain
Solidity
OpenZeppelin Contracts
Ethereum Sepolia Testnet
Wallet Integration
MetaMask

⚙️ Installation

Clone the repository:

git clone https://github.com/rocksun1983/EventTicketNFT-project.git
cd EventTicketNFT-project

Install dependencies:

npm install
🔧 Configure Environment Variables

Create a .env file and add:

PRIVATE_KEY=YOUR_PRIVATE_KEY
SEPOLIA_RPC_URL=YOUR_SEPOLIA_RPC_URL
ETHERSCAN_API_KEY=YOUR_ETHERSCAN_API_KEY
